# EMS
Employee Management System

## Getting started

1. Clone this repository
2. Run `yarn install` , all required components will be installed automatically

   ### iOS

   1. Run `pod install` from `EMS/ios` folder
   2. `yarn start`
   3. `yarn ios`

   ### Android

   no steps required

   1. `yarn start`
   2. `yarn android`
